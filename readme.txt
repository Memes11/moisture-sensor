
This is the Arduino code for a moisture sensor based on an ESP8266.

The hardware is based on an ESP8266 and a simple moisture sensor. it is battery powered.

Hardware list
- ESP8266 module
- Moisture Sensor
- Battery
- Battery charge controller for charge by USB
- DC-DC regulator (to provide proper 3V3 to ESP8266)
- Enclosure (a simple plastic box)
